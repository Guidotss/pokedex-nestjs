# Ejecutar en desarrollo

1. Clona el repositorio
2. Ejecutar
  ```bash
  $ npm install
  # or 
  $ yarn install
  ```

3. Tener Nest CLI instalado
  ```bash
  $ npm i -g @nestjs/cli
  # or
  $ yarn global add @nestjs/cli
  ```

4. Levantar la base de datos
  ```bash
  $ docker-compose up -d
  ```
